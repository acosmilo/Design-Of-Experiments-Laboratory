# Design-Of-Experiments-Laboratory
This repo is a help for students on Industrial Engineer to understand about DOE basics. Since basic hypothesis test until 2^k designs. I used CLAUDE to make the code. I use the theory concept of Montgomery books about DOE. I make the activities of labs with a new problems of my own.

# Index
- LAB_DOE_1 : 1 Sample Z test AND 1 Sample t test
- LAB_DOE_2 : 2 Sample t test (Paried AND Independent)
- LAB_DOE_3 : 1 Factor Design (One way ANOVA, R^2, R^e adj AND Subjective Statements)
- LAB_DOE_4 : 1 Factor Design  (Objective Statements)
- LAB_DOE_5 : 1 Factor and 1 Block Design (ANOVA Hypothesis Test for Treatment and Block)
- LAB_DOE_6 : Box-Cox Transformation AND Post ANOVA (Post Hoc): Tukey (HSD) AND Fisher (LSD)
